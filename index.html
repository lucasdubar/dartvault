🎯 Autodarts

Autodarts est un système de scoring automatique par caméras. Trois caméras USB filment la cible et détectent où chaque fléchette se plante — plus besoin de saisir manuellement !

📦 Matériel nécessaire

Un mini-PC (ex: HP T630), 3 caméras USB (OV9732), un anneau lumineux (Winmau Plasma), et des supports imprimés 3D. Le tout connecté en Ethernet ou WiFi sur le même réseau que votre téléphone/tablette.

⚙️ Configuration

1. Installez Autodarts sur votre mini-PC (autodarts.io)

2. Calibrez les 3 caméras via le Board Manager

3. Activez Autodarts ici dans les réglages et entrez l'IP de votre mini-PC

4. Important : acceptez le certificat HTTPS en visitant https://[IP]:3181 une fois sur chaque appareil

5. Cochez 🎯 Autodarts dans les options de chaque jeu — c'est prêt !

💡 Astuces

• La pastille verte en haut à droite indique la connexion. Tapez dessus pour reset le board si ça bloque.

• La validation nécessaire est forcée en mode Autodarts pour que le retrait des fléchettes déclenche le passage au joueur suivant.
